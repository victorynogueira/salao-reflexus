# Salão Reflexus - Sistema de Gestão PWA

Sistema completo de gestão e agendamento para o Salão Reflexus.

## Funciona como Aplicativo de Celular

Este sistema é um **PWA (Progressive Web App)** - funciona como um app nativo no celular!

## Como Instalar no Celular

### Android (Chrome)
1. Acesse o site no Chrome
2. Toque no **menu (⋮)** no canto superior direito
3. Selecione **"Adicionar à tela inicial"**
4. Confirme o nome e toque em **"Adicionar"**
5. O ícone do Reflexus aparecerá na tela inicial!

### iPhone (Safari)
1. Acesse o site no Safari
2. Toque no botão **Compartilhar** (quadrado com seta para cima)
3. Role para baixo e toque em **"Adicionar à Tela de Início"**
4. Confirme o nome e toque em **"Adicionar"**
5. O app aparecerá como um ícone na sua tela inicial!

### Computador (Chrome/Edge)
1. Acesse o site
2. Clique no ícone de **instalação** na barra de endereço
3. Clique em **"Instalar"**

## Instalação (Desenvolvedor)

```bash
# Instalar dependências
cd salao-reflexus
call npm install

# Iniciar servidor
call npm run dev
```

Acesse: **http://localhost:3000**

**Login padrão:**
- Email: `admin@reflexus.com`
- Senha: `admin123`

## Vantagens do PWA

- Funciona como app nativo
- Ícone na tela inicial
- Abre em tela cheia
- Funciona offline (parcialmente)
- Notificações push
- Não ocupa espaço na Play Store

## Funcionalidades

- Dashboard com gráficos
- Cadastro de clientes
- Cadastro de serviços
- Agenda inteligente (diária/semanal/mensal)
- Agendamento completo
- Controle financeiro
- Modo escuro
- Notificações via WhatsApp
